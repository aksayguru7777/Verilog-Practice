# Verilog Practice – 20 Days Challenge

This repository contains my 20 Days Verilog Practice Challenge where I implement fundamental digital circuits using Verilog HDL.

The goal of this challenge is to strengthen my understanding of Digital Design and improve my hardware design skills.



# Topics Covered

* Half Adder
* Full Adder
* Multiplexer
* Decoder
* Encoder
* Priority Encoder
* Comparator
* Sequential Circuits (coming soon)


# Tools Used
* Verilog HDL
* EDA Playground
* Git & GitHub

# Goal
By completing this challenge, I aim to:
* Improve Verilog coding skills
* Understand digital circuit design
* Build a strong hardware design portfolio
